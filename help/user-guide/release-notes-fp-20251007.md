# Versionshinweise für Feature Pack 20251007 {#release-notes-for-screens-feature-pack}

>[!CAUTION]
>Adobe empfiehlt eine Aktualisierung auf die neueste Version von Adobe Experience Manager (AEM 6.5). Die neuesten Versionsinformationen finden sich [hier](https://experienceleague.adobe.com/de/docs/experience-manager-65/content/release-notes/release-notes).
>FP11.7 mit SP(servicepack) Version 22 oder höher verwenden.

## Verfügbarkeit {#availability}

Das AEM 6.5 Feature Pack 11,7 wurde für AEM Screens veröffentlicht.

Das neueste Feature Pack für AEM Screens 6.5.11.7 steht auf dem [Software Distribution-Portal](https://experience.adobe.com/#/downloads/content/software-distribution/de/aem.html) zum Download zur Verfügung. Navigieren Sie zur Registerkarte **Adobe Experience Manager** und suchen Sie nach **Screens**, um das neueste Feature Pack mit dem Namen **AEM 6.5 Screens FP11.7&rbrace;**.

## Veröffentlichungsdatum {#release-date}

Das Veröffentlichungsdatum für AEM Screens Feature Pack 20251007 ist der 7. Oktober 2025.

### Neue Funktionen {#what-is-new}

* Diese Version beinhaltet nur Sicherheits-Fehlerbehebungen.
